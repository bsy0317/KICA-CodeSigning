# KICA-CodeSigning
공인인증서를 이용한 프로그램 코드서명

범용, 증권용 공인인증서를 이용하여 프로그램에 코드서명을 적용 할 수 있습니다.

![](../header.png)

## 실행방법
```sh
VisualStudio 를 통해 컴파일 한 이후 실행
#openssl 라이브러리를 필요로 합니다.
```
